# Tampermonkey Scripts

Collection of Tampermonkey userscripts.

## Scripts

- `github-redesign.user.js` — GitHub UI enhancements
- `zima-theme.js` — ZimaOS glass theme + auto-login
- `Zima Web Terminal Injector-1.1.js` — ZimaOS terminal injector
- `Factor - Hide Sold Out (Stable)-1.0.js` — Factor 75 meal service helper

## Usage

Install Tampermonkey, then add any `.user.js` file as a new script.
